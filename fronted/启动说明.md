# 前端启动说明

## 前置要求

需要安装 Node.js（包含 npm）

### 安装 Node.js

1. **下载 Node.js**
   - 访问：https://nodejs.org/
   - 下载 LTS 版本（推荐）
   - 选择 Windows Installer (.msi)

2. **安装步骤**
   - 运行下载的 .msi 安装程序
   - 按照安装向导完成安装（默认会添加到 PATH）
   - **重要**：安装完成后需要**重启 PowerShell 或命令提示符**

3. **验证安装**
   打开新的 PowerShell 窗口，运行：
   ```powershell
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功。

## 启动前端服务

### 1. 安装依赖（首次运行需要）
```powershell
cd fronted
npm install
```

### 2. 启动开发服务器
```powershell
npm run dev
```

### 3. 访问应用
打开浏览器访问：http://localhost:3000

## 常见问题

### npm 命令无法识别
- 确保已安装 Node.js
- 重启 PowerShell/终端
- 检查环境变量 PATH 是否包含 Node.js 路径

### 端口被占用
如果 3000 端口被占用，Vite 会自动尝试其他端口（如 3001、3002 等）

### 依赖安装失败
- 检查网络连接
- 尝试使用国内镜像：
  ```powershell
  npm config set registry https://registry.npmmirror.com
  ```
